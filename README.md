# simple_php_search
This repository contain simple php search tool. jQuery, Ajax, Javascript, php are used. This search works in real time. You can see this in the animation below. There is test database in repository.

##Animation search:
![example_search.gif](example_search.gif)

##Install
1. clone repository in domen folder of local server, add local domen (I use Open Server)
2. add database in local server
3. connect the database with db.php
4. turn on the local website
